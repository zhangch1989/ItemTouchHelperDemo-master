# 主要是功能及说明
此项目主要是搜集一些不长用的功能demo以备不时之需，基本上源于网络资源

demo说明：</br>
1.使用ItemTouchHelper实现今日头条 网易新闻 的频道排序、频道移动</br>
2、增加viewpager的多样化展示，实现不一样的轮播效果（来自洋神）；</br>
3、增加了仿百度高德的搜索列表实现抽屉式的列表展示功能；
